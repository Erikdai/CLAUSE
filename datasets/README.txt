Dataset Directory - Data Source Information
==========================================

This directory contains the datasets used for knowledge graph reasoning and question answering.

Dataset Acquisition:
Please download the required datasets from the following link:

https://github.com/yuyuz/MetaQA
https://github.com/aukhanee/FactQA
https://hotpotqa.github.io/

Note: FactQA and HotpotQA datasets require using extract_kg.py to first extract questions and triples before use with the CLAUSE system.
